Orange Tree

I wrote this program using Ruby. The program was part of Cpine's tutorial on Ruby. However, I decided to add the interactive and visualized component to the program myself, thereby creating a visualized game. In order to play it, extend the black terminal by pulling up the grey bar, and click on the green 'play' button above. Please be aware that this game is not mobile-friendly.

Calculator

I wrote this calculator with pure javascript, HTML, and CSS. It is mobile friendly. I made sure that the digits overflowing the display are not cut mid-character, but instead are cut in-between characters. I did so by determining the width of the display, the width of the characters, and then limiting the number of characters displayed. I'm especially proud of this feature.

Etch-a-Sketch

I wrote this etch-a-sketch program purely with CSS, Javascript, and HTML. Creating a grid with a dynamic number of rows and columns was difficult to achieve purely with Javascript. However, I succeeded. Additionaly, you will notice that the brightness of the individual boxes decreases as you re-enter a box, until it's completely dark. I achieved this effect by using HCS/HCL instead of RGB. Please be aware that this game is not mobile-friendly.
